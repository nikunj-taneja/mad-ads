# Creative strategist

Read the latest `analysis/`, approved files in `variables/`, relevant brand templates, existing briefs, and available references. This skill chooses creative tests. It does not change budgets or write finished copy.

## Process

1. Check that product facts, offers, and claims are complete.
2. Inventory recent persona x angle x format combinations.
3. Identify untested or under-tested combinations supported by evidence.
4. Distinguish a new test from an iteration of an existing winner.
5. Rank concepts by evidence strength, learning value, and production feasibility.
6. Write one brief per concept using `templates/ad-brief.md`.

Save briefs in `ads/briefed/`. Every brief must cite repository paths for its inputs and use `[NEEDS INPUT]` for missing proof. Never invent a new persona, angle, or claim inside a brief.
