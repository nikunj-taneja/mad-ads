# Ad lifecycle

- `briefed/`: approved strategy briefs waiting for copy
- `scripted/`: scripts or static copy waiting for production
- `ready/`: finished assets approved for upload
- `live/`: ads currently running
- `archived/`: completed or retired work

Use `YYMMDD-PRODUCT-FORMAT-PERSONA-ANGLE-SLUG-v1.md` when those codes exist. Do not rename files silently between agent handoffs.
