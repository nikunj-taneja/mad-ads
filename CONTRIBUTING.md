# Contributing

Keep contributions brand-neutral and useful without private context. Do not submit real customer records, exports, credentials, ad account IDs, generated competitor media, private URLs, or unlicensed assets.

New skills belong in `skills/<name>/SKILL.md`. New recurring utilities belong in `tools/<name>/` and require a registry entry. Label partial integrations and list the missing configuration or verification step.

Before opening a change, run the included tools and search the full tree for your own client names, domains, emails, account IDs, and absolute home-directory paths.
